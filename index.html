Proyecto: FAQ Bot RAG

Asistente de preguntas frecuentes para una empresa. Usa RAG con un índice de documentos de FAQ y flujo de n8n para ingestión y consulta. Python procesa texto, limpia datos y crea embeddings. Responde en lenguaje natural con citas a las fuentes.

  • Pipeline: carga CSV/PDF → limpieza → embeddings → vector DB → consulta + generación.
  • n8n: cron, webhooks, control de versiones y logs de ejecución.
  • Valor: reduce tiempos de soporte y estandariza respuestas.
  • Próximo: panel simple para subir nuevas FAQs y reindexar.
